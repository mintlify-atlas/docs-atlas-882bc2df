The Lando Drupal plugin is the official integration between [Lando](https://lando.dev) and [Drupal](https://www.drupal.org/), giving you a fully containerized local development environment with zero global dependency management. Choose your Drupal version, PHP version, web server, and database backend — all from a single `.lando.yml` file.

## What the plugin provides

The Lando Drupal plugin configures a complete local stack for Drupal development — no need to install PHP, Composer, or a database server on your machine.

  <Card title="Multi-version support" icon="layer-group">

## Quick example

Add a recipe to your `.lando.yml` and start developing:

```yaml .lando.yml
name: my-drupal-site
recipe: drupal11
config:
  php: '8.3'
  via: apache:2.4
  database: mysql:8.0
```

Then run `lando start` and your Drupal environment is ready.
