The Lando Drupal plugin is maintained by a small team and supported by an active open-source community. This page covers the available support channels — from free community help to dedicated priority support.

## Community support

The fastest way to get help is to join the Lando community on Slack. You can ask questions, share what you're building, and get advice from other Lando users and the core maintainers.

## Reporting bugs and requesting features

If you've found a bug or have a feature request, please open an issue on GitHub. Use the issue template chooser to pick the right category — this helps the maintainers triage and respond faster.

<Tip>
  Before opening a new issue, search the existing issues to see if the problem has already been reported or the feature has already been requested.
</Tip>

## Priority support

If your team needs guaranteed response times, expedited bug fixes, or help building out custom functionality, contact the maintainers directly to discuss priority support options.
